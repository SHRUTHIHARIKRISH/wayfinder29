# WayFinder - TripAdvisor Clone

WayFinder is a responsive travel website inspired by the official TripAdvisor platform. It is built using HTML5 and CSS3, featuring a modern and user-friendly interface. The website includes pages for Hotels, Flights, Restaurants, About, and Contact, along with a search section, filters, destination cards, and responsive layouts using Flexbox and CSS Grid.

## Features
- Responsive design for all devices
- Modern navigation bar with search
- Featured destinations, hotels, and restaurants
- Search and filter sections
- Interactive card layouts
- Clean and professional UI
- Built using HTML5 and CSS3
